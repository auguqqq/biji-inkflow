
# 笔纪 Inkflow Studio v2.0.0

**专注中文创作的现代化码字助手**

笔纪是一款专为网络文学作者和中文写作者设计的 Web 应用程序。它结合了极简主义的视觉设计、强大的自动排版功能、沉浸式的小黑屋模式以及基于大语言模型（Gemini/DeepSeek/OpenAI）的智能创作辅助，致力于让文字流淌，让灵感生花。

## ✨ 核心功能 (近期更新)

### 🧐 深度精修与批注
- **卡片式批注**：不同于泛泛而谈的聊天，AI 现在会像人类编辑一样，逐段扫描你的草稿。
- **问题定位**：针对“背景冗余”、“对话悬浮”、“逻辑硬伤”等问题生成具体的修改建议卡片。
- **点击跳转**：点击批注卡片，自动高亮并定位到正文对应的段落位置（支持模糊搜索定位）。

### 📚 长短篇分流管理
- **双模创作**：新建作品时可选择「长篇小说」或「短篇文集」。
- **差异化体验**：
    - **长篇**：侧重连载、伏笔管理、宏大叙事，AI 辅助黄金三章与节奏把控。
    - **短篇**：侧重脑洞、反转、快节奏，AI 辅助黄金前三句与情绪价值爆发。
- **题材细分**：内置“玄幻、都市、古言、科幻、规则怪谈”等 17 种网文主流题材，AI 将根据题材调整建议风格。

### 🤖 深度 AI 责编体系
- **三位一体**：
    - **毒舌主编 (Critic)**：深度复盘剧情，扫描毒点，评分打分。
    - **灵感搭档 (Partner)**：卡文时陪聊，提供脑洞与后续剧情推演。
    - **润色专家 (Polisher)**：优化文笔，增强画面感与代入感。
- **智能分析报告**：生成全书/单篇的情绪曲线 (Sentiment Arc)、五维雷达图 (剧情/文笔/深度/人物/节奏) 及高频词云。
- **智能捉虫**：一键扫描错别字、语病，支持点击定位修正。
- **强制防乱码**：底层 System Prompt 强化指令，杜绝 AI 输出外语或乱码，确保 100% 简体中文交互。

### 🎧 沉浸式创作环境
- **白噪音混音台 (Pro)**：内置高清雨声、风声、篝火、图书馆等环境音，支持多音轨混音。
- **自定义音效**：支持上传自己的 MP3 文件（如机械键盘声）与环境音混合。
- **暗黑模式优化**：全界面（包括侧边栏大纲、编辑器、统计图表）完美适配深色主题。
- **视觉优化**：
    - **柔和高亮**：优化了选中文字的高亮颜色，白/暗主题下均不再刺眼。
    - **智能交互**：优化了润色流程，点击润色按钮后会跳转至对话框并预填内容，给予作者二次确认的机会。

## 🏗️ 基础功能

*   **沉浸式体验**：支持多种护眼主题（象牙白、水墨黑、护眼绿等）。
*   **智能排版**：一键自动缩进、段落优化，符合中文出版与网文排版规范。
*   **小黑屋模式**：强制锁定屏幕，设定字数或时间目标，杜绝摸鱼。
*   **多书架管理**：支持导入 TXT/Markdown，自动识别章节分卷。
*   **数据统计**：可视化展示每日码字量、连续打卡天数及周产量趋势。
*   **版本回溯**：自动保存章节历史版本，随时一键还原，防止误删。
*   **本地优先**：所有数据（除 AI 交互外）均存储在本地浏览器 IndexedDB/LocalStorage，保障隐私安全。

## 🔐 安全与隐私说明 (Security & Privacy)

1.  **代码安全**：源码不含真实 API Key。
2.  **API Key 管理**：
    *   **BYOK 模式**：用户在“设置”中输入的 API Key 仅保存在本地浏览器，**绝不会**上传至服务器。
3.  **防盗刷**：建议在 AI 服务商后台设置配额上限。

## 🛠️ 部署与安装

本项目是基于 React + TypeScript + Tailwind CSS 构建的纯前端应用。

### 本地运行

1.  安装依赖：`npm install`
2.  启动：`npm start`

### 静态部署

1.  构建：`npm run build`
2.  将 `build/` 目录上传至 Vercel/Netlify 或任何静态服务器。
